# Password Security Labs

This repository documents my cybersecurity learning journey as I study password security, authentication, and account protection.

## Topics Covered

* Password Hashing
* Salting
* Multi-Factor Authentication (MFA)
* Password Managers
* Authentication Concepts

## Goals

* Learn core cybersecurity concepts
* Practice technical documentation
* Build a cybersecurity portfolio
* Prepare for entry-level SOC and cybersecurity roles

## Disclaimer

All content is created for educational purposes only. No real passwords, credentials, or sensitive information are included.
